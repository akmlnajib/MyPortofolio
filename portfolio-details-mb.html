MicroBooks

MicroBooks is a web-based platform that allows users to access digital book collections and interact via a built-in chat feature. This project was built with modern technology using Laravel as the backend framework, as well as a combination of JavaScript, TypeScript, and Blade for dynamic frontend development. Key features include digital book management and real-time communication between users to enhance the reading experience and collaboration.
