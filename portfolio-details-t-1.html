Waze-User-Churn-Prediction-EDA

This project aims to analyze Waze application user data to predict potential churn (discontinuation of service use). Using Exploratory Data Analysis (EDA) techniques, visualizations created in Tableau help identify user behavior patterns, factors contributing to churn, and provide insights for user retention improvement strategies. These interactive visualizations enable more effective data-driven decision making by exploring relationships between variables such as application usage frequency, session duration, and other user activity.
