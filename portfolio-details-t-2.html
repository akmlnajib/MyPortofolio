TikTok End of Course Project

This project is part of a data analytics study to understand TikTok user patterns. Using Tableau, comprehensive data analysis was performed to illustrate key insights into user behavior, engagement trends, and content performance. Interactive visualizations help highlight key metrics such as user retention rates, average watch time, and the most popular types of content. The results of this analysis can be used to improve marketing strategies, improve recommendation algorithms, and improve user experience on the TikTok platform.
